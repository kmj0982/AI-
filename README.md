# AI 도구 활용 및 비교 분석 프로젝트

## 📌 프로젝트 소개
본 프로젝트는 다양한 인공지능(AI) 도구를 활용하는 방법을 학습하고, 대표적인 AI 서비스인 ChatGPT와 Gemini를 비교 분석하기 위해 진행되었다. 특히 프롬프트 엔지니어링을 중심으로 실제 활용 사례를 분석하고, GitHub 협업 환경에서 AI를 효과적으로 사용하는 방법을 탐구하는 것을 목표로 한다.

최근 AI 기술은 과제 작성, 코딩, 번역 등 다양한 분야에서 활용되고 있으며, 올바른 사용 방법에 따라 결과의 품질이 크게 달라진다. 따라서 본 문서는 AI 도구의 개념부터 실전 활용, 그리고 협업 방법까지 체계적으로 정리하였다.

### 💡이 가이드의 구성 및 주요 내용
본 문서는 AI 도구의 기초부터 실전 협업 활용까지 단계별로 학습할 수 있도록 구성되었다.
* 프롬프트 엔지니어링(prompt-engineering): 좋은 답변을 얻기 위한 프롬프트 작성 전략 및 비교 실험
* 실전 유스케이스(use-case): 과제 작성, 코딩, 번역 등 실제 사례 기반의 활용법 제시
* 모델 비교 분석(comparison): 대표적 도구인 ChatGPT와 Gemini의 장단점 및 특징 비교
* GitHub 기반 협업(collaboration): 팀 프로젝트와 개발 환경에서 AI를 활용한 효율적인 협업 방식
* 실전 꿀팁(ethics-and-limits): 작업 속도를 높여주는 단축키 및 유용한 부가 기능 및 주의사항 정리
