# LLM 기반 AI의 활용 사례
최근 몇 년 사이 Large Language Model(LLM)을 기반으로 한 AI 기술은 다양한 산업과 일상 속으로 빠르게 확산되고 있다. ChatGPT나 Gemini 같은 서비스는 단순한 챗봇을 넘어, 실제 업무와 생산성을 혁신하는 도구로 자리 잡았다. 본 파일은 LLM 기반 AI의 주요 use case를 설명하기 위해 작성되었다.
---
## 1. 교육 및 학습 보조
학업 경쟁이 치열해지면서 LLM은 개인화된 학습 도우미로 주목받고 있다. 
* 개념 설명 및 요약
* 문제 풀이 가이드
* 맞춤형 학습 콘텐츠 제공
* 외국어 학습 지원
학생의 수준과 질문에 맞춰 설명을 조정할 수 있어, 기존 교육 방식의 한계를 보완한다.
